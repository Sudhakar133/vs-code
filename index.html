Difference between HTTP1.1 vs HTTP2

HTTP1.1
1.There is head of line blocking that blocks all the requests behind it until it doesn’t get its all resources.
2. It uses requests resource Inlining for use getting multiple pages
3. It compresses data by itself.


HTTP2
1.It allows multiplexing so one TCP connection is required for multiple requests.
2.It uses PUSH frame by server that collects all multiple pages
3. It uses HPACK for data compression.

objects and its internal representation in Javascript

1)Objects:
The following code assigns a simple value (Mercedes) to a variable named car:
var car = "Mercedes";
Objects are variables too. But objects can contain many values.
The following code assigns many values (Mercedes, C-class, White and soo on) to a variable named Car:
var car = {Make: “Mercedes”, Model: “C-Class”, Color: “White”, Fuel: Diesel, Weight: “850kg”, Mileage: “8Kmpl”, Rating: 4.5};
The values are written as name:value pairs
Syntax:
var = {key1: value1, key2: value2,... keyN: valueN};
So, conclusion and definition for JS objects is “JavaScript objects are containers for named values”.
